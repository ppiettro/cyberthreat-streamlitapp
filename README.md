## English

### About the Project
This project is currently under active development. Its main purpose is to collect, process, analyze and display cyber threat data. The tool aims to provide an easy way to access and visualize specific threat information.
The application is built using Python + HTML & CSS and deployed via the **Streamlit** framework, allowing users to access the data dashboards directly through a web browser.
The data is relative to **Brazil** threat landscape.


### ⚠️ Disclaimer & Data Source (ALL data collected from publicly available, belonging to **Kaspersky**)

#### **Important:** **All data**, metrics, and information displayed or processed by this software are retrieved directly from a **public website belonging to Kaspersky**.
This project **does not** own the data, **nor does it** claim authorship over the security intelligence provided. It serves as a tool/interface to access publicly available information and to provide an alternative way to visualize and interact with the information.

#### Data links:
1. https://cybermap.kaspersky.com/stats#country=215&type=KAS&period=w
2. https://threats.kaspersky.com/

### ✨ Features (Planned/Implemented)
- **Data (Web Scrape) Collection:** Collects the last week data from public sources.
- **Top Threat Aggregator:** Collects the top 10 threats of each detection type on the Kaspersky Cybermap Statistics.
- **Web Accessibility:** Easy access via a Streamlit-deployed URL.
- **Data Visualization:** Interactive charts, metrics and dashboard regarding cybersecurity threats to explore from different perspectives.
- **CSV & JSON Exporter:** Exports the statistic data to CSV and the informational data to JSON.
- **Interactive Threat Explorer:** Has a selectbox to search a specific threat and be redirected to the official kaspersky threats website to get more information.

# Português BR

### Sobre o Projeto
Este projeto encontra-se atualmente em desenvolvimento ativo. Seu principal objetivo é coletar, processar, analisar e exibir dados sobre ameaças cibernéticas. A ferramenta visa a fornecer uma maneira fácil de acessar e visualizar informações específicas sobre ameaças.
A aplicação é construída utilizando Python + HTML & CSS e seu deploy é realizado através do framework **Streamlit**, permitindo que os usuários acessem os painéis de dados diretamente através de um navegador web.
Os dados são relativos ao cenário de ameaças do **Brasil**.

### ⚠️ Aviso Legal e Fonte de Dados (TODOS os dados coletados de fontes públicas, pertencentes à **Kaspersky**)

#### **Importante:** **Todos os dados**, métricas e informações exibidas ou processadas por este software são obtidos diretamente de um **site público pertencente à Kaspersky**.
Este projeto **não** possui a propriedade dos dados, **nem** reivindica autoria sobre a inteligência de segurança fornecida. Ele serve como uma ferramenta/interface para acessar informações publicamente disponíveis e para fornecer uma maneira alternativa de visualizar e interagir com essas informações.

#### Links dos dados:
1. https://cybermap.kaspersky.com/stats#country=215&type=KAS&period=w
2. https://threats.kaspersky.com/

### ✨ Funcionalidades (Planejadas/Implementadas)
- **Coleta (Web scrape) de Dados:** Coleta os dados da última semana em fontes públicas.
- **Agregador das Principais Ameaças:** Coleta as 10 principais ameaças de cada tipo de detecção nas Estatísticas do Cybermap da Kaspersky.
- **Acessibilidade Web:** Acesso facilitado via URL com deploy no Streamlit.
- **Visualização de Dados:** Gráficos interativos, métricas e painéis sobre ameaças de cibersegurança para explorar sob diferentes perspectivas.
- **Exportador CSV e JSON:** Exporta os dados estatísticos para CSV e os dados informativos para JSON.
- **Explorador Interativo de Ameaças:** Busque uma ameaça específica e seja redirecionado para o site oficial de ameaças da Kaspersky para obter mais informações.
